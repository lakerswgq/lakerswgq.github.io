webp first experience

5月 04, 2017 发布在 front end, others

WebP是Google推出的图片格式，同时支持有损和无损压缩，还支持动画，压缩比又高，有种RMB玩家的感觉。

WebP介绍

  • 维基
  • 知乎
  • ISUX

WebP在很多方面吊打png, gif, jpg,但还是有些许不足，比如

  • 编码、解码速度慢

    WEBP与JPG相比较，编码速度慢10倍，解码速度慢1.5倍

  • 兼容性

    • Can I Use

不过比起图片体积的大幅减少，即使解码速度会慢一点页面性能还是会得到提高。

近期打算把使用七牛作为博客的图床，hexo有一个插件，七牛提供了一些方便的图片处理样式，其中就有转webp的。

在七牛的 存储空间>图片样式 下选择新建图片样式，选择你想要的图片处理样式，配置一番，
之后可以起个名字,在hexo七牛插件的extend配置项下加上就可以了。
